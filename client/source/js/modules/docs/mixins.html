Mixins

cosmetic($module, $values)
cosmetic($module, $property, $values)

Creates a cosmetic module for setting a single property.

A CSS property name by default is supposed to be the same as the module name.

The $values could be either list or map. In the first case, resulting class name will be formed by appending list item itself to the module name.

@include cosmetic('float', (left, right));
.float-left {
  float: left !important;
}

.float-right {
  float: right !important;
}

Otherwise a class name will be a module name conjuncted with an item key, a value will be taken from map.

@include cosmetic('margin', ('m': 32px, 'l': 128px));
.margin-m {
  margin: 32px !important;
}

.margin-l {
  margin: 128px !important;
}

If a property name needs to be different from a module name, it can be passed as an optional second argument.

@include cosmetic('valign', 'vertical-align', (bottom, top));
.valign-bottom {
  vertical-align: bottom !important;
}

.valign-top {
  vertical-align: top !important;
}

font-face($family, $filename, $weight: normal, $style: normal)

Allows easier custom fonts integration.

@include font-face('Open Sans', 'OpenSans-BoldItalic', bold, italic);
@font-face {
  font-family: 'Open Sans';
  src: url("/assets/fonts/OpenSans-BoldItalic.eot");
  src:
    url("/assets/fonts/OpenSans-BoldItalic.eot?#iefix") format('embedded-opentype'),
    url("/assets/fonts/OpenSans-BoldItalic.woff") format('woff'),
    url("/assets/fonts/OpenSans-BoldItalic.ttf") format('truetype'),
    url("/assets/fonts/OpenSans-BoldItalic.svg#Open+Sans") format('svg');
  font-weight: bold;
  font-style: italic;
}

font-height($font-size, $line-height)

Shorthand notation for setting font metrics of an element.

@include font-height(12px, 1.62);
font-size: 12px;
line-height: 1.62;

hidpi-image($image-path)

Includes background-image and background-size for the named HIDPI image.

@include hidpi-image('flower-320x160.jpg');
width: 160px;
height: 80px;
background-image: url('/assets/images/flower-320x160.jpg');
background-size: 160px 80px;

hidpi-inline($image-path)

Embeds background-image and includes background-size for the provided HIDPI image.

@include hidpi-image('flower-320x160.jpg');
width: 160px;
height: 80px;
background-image: url('data:image/jpg;base64,...');
background-size: 160px 80px;

image-size($image-path, $factor: 1)

Sets the width and the height of an element to the dimensions of the provided image.

The second argument is a divisor which is applied onto both dimensions.

@include image-size('flower-320x160.jpg');
width: 320px;
height: 160px;

position($position, $coordinates: none)
position($coordinates)

Shorthand notation for setting position of an element.

The first argument is optional and sets position of an element (absolute, relative, etc.).

The second argument is a space delimited list of offsets (top, right etc.) that follows the standard CSS shorthand notation.

Offsets with values of none are omitted.

@include position(absolute, 12px 0 none);
position: absolute;
top: 12px;
right: 0;
left: 0;

reject($property, $value, $filters)

Includes property only if its value is not in the list of filters, is useful for creating mixins.

/* first example outputs nothing */
@include reject(width, 120px, 60px 120px 240px);
@include reject(width, 20px, 60px 120px 240px);
/* first example outputs nothing */
width: 20px;

size($size)

Shorthand notation for setting size of an element.

The only argument is a space delimited width and height. If height is omitted, it is considered to be equal to width.

Dimensions with values of none are omitted.

@include size(100%);
width: 100%;
height: 100%;
@include size(100px 50px);
width: 100px;
height: 50px;
