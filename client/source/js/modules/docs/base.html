Base layer

Base or first layer is a core part of an interface. It is based on most reusable and abstract constructions:

  • forms,
  • buttons,
  • navigation blocks, etc.

Base layer styles must be integrated with designer’s style guides as close as possible. As modules of first layer are meant to be reused across all website interfaces, they must look appropriate and fit to other interface parts without modification.
