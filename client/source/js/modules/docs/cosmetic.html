Cosmetic layer

Cosmetic or third layer consists of simple, slightly affecting styles:

  • text colors,
  • low-level OOCSS – few properties per CSS class (e.g. margin-top-xl, width-l),
  • global modifiers, etc.

Layer can be omitted in some cases of methodology usage, but in big projects, this layer allows to solve the duplicate styling issue and to describe rare, non-project conditions, going DRY.

Float

clear-(value)

Sets clear property of an element.

(value) should be either none, left, right or both.

float-(value)

Sets float property of an element.

(value) should be either none, left or right.

Sizes

width-(value)
height-(value)

Sets width or height property of an element.

(value) should be either auto, xxs, xs, s, m, l, xl, xxl or none for zero value.


margin-(value)
margin-(direction)-(value)
padding-(value)
padding-(direction)-(value)

Sets margin or padding property of an element, setting all sides at once (e.g. margin-xs) or one particular side (e.g. margin-top-xs).

(value) should be either xxs, xs, s, m, l, xl, xxl or none for zero value.

(direction) should be either top, right, bottom or left.

Text

text-(alignment)

Sets text-align property of an element.

(alignment) should be either left, center, right or justify.

text-bound

Sets max-width property of an element with a value dependant on it’s font-size. This limits the maximum number of words in a single line to keep better readability.

.text-bound {
  max-width: 48em !important;
}

Vertical-align

valign-(value)

Sets vertical-align property of an element.

(value) should be either baseline, bottom, middle or top.
